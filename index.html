How does javascript work?

Ans: Javascript is a single threaded programming language.Javascript can handle one task at a time. Javascript run on chrome browser or Node.js by v8 engine.First v8 engine take the js source code and parse it and transfer it into a Abstract syntax tree.Then this abstract syntax tree go to interpreter ignition and ignition optimized it into Byte code by using just in time compiler for executed the code very fast.And then Byte code converted into Machine code by turbofan of v8 engine.Finally codes runs on the cpu.

Difference between local storage vs session storage?

Ans: Local storage save data permanently but in Session storage save data temporary.We have to clear local storage manually but when the browser last tab is closed session storage is clear automatically.If we set a key , value in local storage and close the browser and again open the browser the key ,value will save in local storage. But session storage do no save the value we set.Local storage is shared between all tabs but session storage is not shared between all tabs.Local storage can be hacked but session storage can not be hacked
